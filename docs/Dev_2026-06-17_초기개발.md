## 4. Supabase 연동 내역
- 클라이언트: `src/lib/supabase.js` — `createClient`, 세션 유지/자동갱신, `detectSessionInUrl`
- 접두사 상수 `PREFIX='webpro_'`, 테이블 매핑 `T = { profiles, progress, enrollments }`
- 인증: `hooks/useAuth.jsx`
  - `AuthProvider` 로 세션 구독(`onAuthStateChange`), 로그인 시 프로필 upsert
  - `signIn('google'|'kakao')` → `signInWithOAuth`, redirect `/auth/callback`
- 진도: `hooks/useProgress.jsx` + `lib/db.js`
  - 섹션 단위 완료를 낙관적 업데이트 후 `webpro_progress` upsert
  - chapterId·section_no 기준, 트랙/챕터 진도율 계산
- 스키마: `supabase/schema.sql`
  - `webpro_profiles`, `webpro_progress`, `webpro_enrollments`
  - 전 테이블 **RLS 활성화 + 본인 데이터 한정 정책**(auth.uid 기준)

## 5. OG / 공유 미리보기
- `index.html` 에 Open Graph(`og:url/title/description/type/image/site_name`) + Twitter 카드 메타
- `scripts/gen-og.mjs` — **sharp** 로 SVG → `public/og-image.png`(1200×630) 렌더
  - 브랜드 다크 그라데이션 + 로고 + 한글 타이틀 + "16개 챕터 · 62개 강의" 배지
  - 빌드 시 `npm run og` 자동 실행(Actions)
- 카카오 공유 디버거로 검증 권장: https://developers.kakao.com/tool/debugger/sharing

## 6. 배포 구성
- `.github/workflows/deploy.yml` — main 푸시 시
  1. `npm ci` → `npm run og` → `npm run build`(Secrets 주입) → `dist/404.html`(SPA fallback) → Pages 배포
- 커스텀 도메인: `public/CNAME = bootcamp.dreamitbiz.com`, vite `base:'/'`
- 필요 Secrets: `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`
